Włącz ETAP 2 — PROJECT VISION.

Zasady:

- Zaproponuj kompletną wizję projektu (paleta, typografia, styl UI/UX, kolejność sekcji, layout system, mini-wireframe, mikro-interakcje/animacje).
- Opisz wszystko tekstowo + ewentualne szkice ASCII/markdown.
- **Poproś o akceptację** przed przejściem do kodu.
- NIE generuj kodu w tym kroku.

Format propozycji:

1. Paleta kolorów (primary/secondary/accent/neutral + odcienie)
2. Typografia (fonty, skale, hierarchia)
3. Styl UI/UX (minimalistyczny/bold/playful/corporate/etc.)
4. Layout system (Grid/Flexbox/Bento/Hero+Sections/etc.)
5. Kolejność sekcji (mini-wireframe)
6. Animacje/mikro-interakcje (scroll reveal, hover states, transitions)
7. Responsive breakpoints (1024/768/480/360)
